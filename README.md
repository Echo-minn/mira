# Portfolio (static)

This repository contains a simple static portfolio page generated from the resume PDFs in the root folder.

## How to view

- Open `index.html` directly in a browser, or
- Use any static server (optional), e.g.:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html`: the portfolio page
- `styles.css`: styling
- `assets/placeholder-headshot.svg`: headshot placeholder
- `assets/placeholder-project.svg`: project image placeholder
- `Minn_s_Resume.pdf`, `Mira_SWE.pdf`, `Mira_MLE.pdf`: resume PDFs linked from the page
